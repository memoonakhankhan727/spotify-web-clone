 Spotify Web Clone

A Spotify Clone built using **HTML, CSS, and JavaScript**.  
This project replicates the look and feel of Spotify’s web player with playlist cards, play button UI, and responsive design.  
Created as part of web development practice and portfolio building.

---

 Features
- 🎶 Playlist cards with cover images
- ▶️ Play button UI
- 📱 Responsive design for mobile and desktop
- 🎨 Custom styling with CSS
- 🔊 JavaScript-based audio playback


 Tech Stack
- **HTML** – Structure of the web app  
- **CSS** – Styling and layout  
- **JavaScript** – Dynamic functionality and audio playback  



⚡ How to Run
1. Clone the repository:
2. Open the project folder in VS Code.
3.Run index.html in your browser.
